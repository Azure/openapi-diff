### 1030 - DifferentDiscriminator

**Description**: Checks whether value of `discriminator` has changed from the previous specification.

**Cause**: This is considered a breaking change.

**Example**: Value of `discriminator` element of Parameter `Pet` changed from `petType` to `name` in the new version.

Old specification
```json5
{
  "swagger": "2.0",
  "info": {
    "title": "swagger",
    "description": "The Azure Management API.",
    "version": "2016-12-01",
    ...
    ...
  "definitions": {
    ...
    "Pet": {
      "type": "object",
      "discriminator": "petType",
      "properties": {
        "name": {
          "type": "string"
        },
        "petType": {
          "type": "string"
        }
      },
      "required": [
        "name",
        "petType"
      ]
    },
    ...  
```

New specification
```json5
{
  "swagger": "2.0",
  "info": {
    "title": "swagger",
    "description": "The Azure Management API.",
    "version": "2016-12-01",
    ...
    ...
  "definitions": {
    ...
    "Pet": {
      "type": "object",
      "discriminator": "name",
      "properties": {
        "name": {
          "type": "string"
        },
        "petType": {
          "type": "string"
        }
      },
      "required": [
        "name",
        "petType"
      ]
    },
    ...  
```
